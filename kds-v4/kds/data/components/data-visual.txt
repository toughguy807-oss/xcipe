Data Visual

정량적 데이터를 시각적으로 표현하여 빠르게 패턴, 추세, 비교를 이해할 수 있습니다.

그래프에서는 정보의 유형과 목적에 따라 적절한 색상 팔레트를 선택해야 하며, 데이터 간의 구분과 강조를 명확하게 전달해야 합니다.

배합형 Default

범주 구분이 명확한 데이터에 적합하며, 컬러 간 대비가
크기 때문에 강조가 필요한 항목에 효과적입니다.

	
배합형 Light

다수 항목을 부드럽게 구분할 때 사용하며, 강한 시각적 부담 없이 전체 흐름을 전달할 수 있습니다.



	


강조형 strong

기본적으로 배합형과 유사한 구성을 가지되, 특정 데이터를 
상대적으로 강조하고자 할 때 Strong 컬러로 강조합니다.

	
확장형 Extended

데이터 항목이 많거나 범주가 세분화된 경우에 적합하며,
전체 데이터 분포를 확인하는 상황에 배합형과 함께 활용합니다.



	


순차형 Sequential

동일 계열의 색상 명도 단계를 활용해 값의 크기나 흐름을 
자연스럽게 표현할 때 사용합니다.

	










	







Type

데이터를 보는 목적에 따라 적절한 차트를 선택합니다.




1. Bar Chart

카테고리별 데이터 값의 크기를 비교할 때 사용합니다. 비교 항목은 5개 이하로 구성합니다.
가장 일반적인 Vertical과 Horizontal, 그리고 카테고리 내 데이터를 한 번 더 구분한 Dual과 Stacked 4가지 Type을 제공합니다.

2. Pie/Donut Chart

한 범주 내에서 구성 항목을 전체의 비율로 나타냅니다. 주로 가장 큰 값에서 가장 작은 값 순으로 배치하며,
8개 미만의 카테고리를 제공합니다. 카테고리가 8개 이상인 경우, 7개까지만 표시하고 나머지 카테고리는 기타로 합쳐 표시합니다.

3. Line Chart

시간 흐름에 따른 추이를 비교할 때 사용합니다.




Variation
Hidden Legend

범례의 양이 많거나 범례가 상시 노출되지 않아도 되는 경우, 
범례 보기 버튼을 사용합니다.

	
Emphasized Chart

특정 데이터를 강조해야 할 경우, Strong 컬러를 사용해 시각적 
우선순위를 제공합니다.
