Usage
Tokens
Breakpoint

화면 크기와 해상도를 기준으로 반응형 레이아웃을 사용하여 환경에 최적화된 구조로 표시됩니다.




Grid

컬럼, 마진, 거터로 구성되어 있으며, 다양한 디바이스에서 일관된 사용자 경험을 제공합니다.

Tablet
Desktop







Responsive Layout

그리드 시스템을 기반으로 화면 크기에 따라 UI 요소가 유연하게 변경됩니다.




Spacing

스페이싱은 UI 요소 간의 간격과 여백을 정의하여 콘텐츠의 위계와 가독성을 조율합니다.

4pt Grid

모든 여백과 간격을 4point 단위로 설정해 규칙성을 확보합니다. 
상세하고 복잡한 컴포넌트의 경우, 2의 배수 단위를 제한적으로 사용합니다.




Use case

Space 16
