Type

State
1. Text Input

텍스트를 입력해 AI에게 질문하고 지시합니다.

2. Voice Input

AI가 사용자의 음성을 인식하고 텍스트로 변환해 입력합니다.

3. Image Upload 

한 개 이상의 이미지를 업로드할 수 있습니다.

4. File Upload 

한 개 이상의 파일을 업로드할 수 있습니다.

 5. Tool Selector 

도구를 활용해 AI의 기능을 확장하고 복잡한 작업을 수행할 수 있습니다.
