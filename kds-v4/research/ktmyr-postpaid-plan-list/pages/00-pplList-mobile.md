### 3. 서브카테고리 칩 (Sub-tab — gnb 내부)

### 4. 페이지 헤딩
- 텍스트: "후불요금제"
- 스타일: h2 bold 16px / weight 700 / line-height 20.8px / 좌측 정렬

### 5. 카테고리 칩 그리드 (가로 스크롤)
- 5개 칩: `👍 인기 요금제` / `🎁 제휴혜택` / `👵 부모님` / `🧑 청소년` / `♾️ 완전 무제한`
- 칩 디자인: 흰 배경 + 옅은 회색 border + 라운드 12~16px + 이모지 + 짧은 라벨
- 모바일은 가로 스크롤 (3개 정도 viewport 에 노출, 나머지는 우측 슬라이드)
- 데스크탑은 5개 모두 한 줄에 균등 분포

### 6. 검색·필터 행
- **필터 버튼** (좌, 동그라미 형태):
  - 모바일: 다크 네이비 `rgb(55, 59, 89)` = **`#373B59`** 원형 배경 + 흰색 필터 아이콘 (텍스트 없음, 아이콘만)
  - 데스크탑: 같은 다크 네이비 배경의 **알약형(pill)** 버튼 + 필터 아이콘 + "필터" 텍스트
- **검색 input** (우): 흰 배경 + 회색 `#999` 1px border + 라운드 50px (pill) + 좌측에 돋보기 아이콘 + placeholder "검색어를 입력해 주세요."

### 7. 결과 헤더 (정렬 영역)
- 좌: `총 [250]` — 숫자만 cyan-teal `rgb(20, 186, 215)` = **`#14BAD7`** 굵게 강조, `총` 글자는 검정
- 우 (모바일): `추천 순 ↑↓` 단일 드롭다운 (탭하면 옵션 5개 펼침)
- 우 (데스크탑): 5개 옵션 가로 일렬
  - `추천 순 ⓘ` (active — 검정 weight 400) / `인기 순 ⓘ` / `데이터 많은 순` / `요금 낮은 순` / `할인 기간 후 요금 낮은 순`
  - 비활성은 회색 `#777` weight 300
- 두 활성 옵션 옆에는 정보 아이콘 ⓘ (`추천 순`, `인기 순`)

### 8. 요금제 카드 (1개)
- 흰 배경 카드 + 옅은 그림자 + 라운드 큼 (눈대중 16~20px)
- **카드 내부 정보 위계** (위→아래):
  1. 사업자 로고 (예: "toss mobile", "스카이라이프", "EG", "M mobile", "쉐이크" 등 — 14개 이상의 알뜰폰 사업자)
  2. 요금제 이름 — h3 또는 strong / bold 검정 / 약 18px
     - 예: "토스실속150분1GB", "모두 충분 1.5GB+", "토스 마이알뜰 CU100GB+", "EG CU 무제한 100G+5", "쉐이크 LTE 100GB+_네이버페이(5,000)"
  3. **가격 줄** — `월` (검정, 17~20px, weight 700) + `[숫자],[숫자]원` (**cyan-teal `#14BAD7`, 19~22px, weight 700**) + 우측 보조 메타
     - "7개월 이후 4,800원" (회색 보조 텍스트 — 할인 기간 안내)
     - "평생 할인" (회색 보조 텍스트 — 평생 할인 강조용)
  4. **스펙 행** (회색 배경 박스 안 또는 동급 영역) — 가로 3그룹
     - `📊 데이터: 1GB` (또는 "100GB + 최대 5Mbps" 식 상세)
     - `📞 음성통화: 150분` (또는 "무제한")
     - `✉️ 메세지: 100건` (또는 "무제한")
  5. **카드 우상단**: `[+ 비교]` 버튼 (체크박스 형태, 옅은 회색 outline + 회색 텍스트, 누르면 활성화)

### 9. Sticky 비교 영역 (선택 시 등장)
- 페이지 어디서나 카드 우상단 `+ 비교` 를 누르면 화면에 sticky 형태로 등장
- 좌: `비교 대상 요금제` 라벨 + 빨강 원 안 흰색 숫자 `[1]` (선택된 개수, 최대 3)
- 우: `[비교하기]` 버튼 — 큰 사이즈, **`#07BFE0` 채움 + 흰색 텍스트 + 라운드 8px**
- 데스크탑은 가로 막대형, 모바일은 가로 막대형 같지만 풀폭

### 10. 더보기 페이지네이션
- 외곽선 버튼 — 풀폭 흰 배경 + 회색 outline + 검정 텍스트 + `더보기(10/250)`
- 진행되는 형태: `(10/250)` → `(20/250)` → ... 누르면 다음 10개 로드

## Notes (에이전트 해석)
- URL 경로는 `/fe/mypage/ppl/pplList.do` 이지만 실제로는 **공개 후불요금제 상품 목록 페이지** — 마이페이지 안의 "가입한 후불요금제 목록" 이 아니다.
- "마이페이지 안 가입한 요금제 목록" 은 별도 페이지가 있을 가능성이 높으나 로그인 필요로 직접 접근 불가. 이 페이지가 가장 유사한 정보 구조 (요금제 카드 리스트 + 가격 + 스펙).
- **컬러 시그니처가 새로움**: 가격 강조와 비교하기 CTA 모두 **cyan-teal 계열** (`#14BAD7` / `#07BFE0`) — 기존 brand.md 에 기록된 Teal `#499690` (헤더 로고용) 보다 훨씬 밝은 청록. 셀프개통 영역의 Purple Primary 와는 또 다른 컬러 영역. **상품 영역의 brand 시그니처 추정**.
- 필터 버튼 컬러 `#373B59` (다크 네이비) 도 새 컬러. brand 의 회색/검정 베이스에서 약간 채도 있는 변형.
- 250개의 방대한 카드 리스트 + 강한 필터·검색·정렬 + 비교 기능 — **선택 부담 해소 UX** 가 페이지 핵심. brand 의 단계형 큐레이션 UX 모티프와는 다른 **"공급자 정보 풀 노출 + 사용자 필터링"** 패턴.
- 카드 안 사업자 로고 ("toss mobile", "스카이라이프", "EG", "M mobile", "쉐이크") 가 노출 — KT 마이알뜰폰은 자체 브랜드도 있지만 14개 이상 알뜰폰 사업자의 요금제를 통합 노출하는 **마켓플레이스성 사이트**.
