# 리서치 브리프: shop-kt-main

## 개요
- 목적: KT Shop (shop.kt.com) 메인 화면을 KDS 로 재디자인하기 위한 사전 분석
- 시드 URL (mobile): `https://m.shop.kt.com/m/main.do`
- 시드 URL (desktop): `https://shop.kt.com/main.do`
- 모드: single (메인 화면 mobile + desktop 2 viewport 분석)
- 뷰포트 패턴: **적응형 (Adaptive — 별도 모바일 URL `m.shop.kt.com` 보유. mobile / desktop URL·레이아웃 분리)**
- 시안 대상 viewport: 사용자 결정 필요 (보통 모바일 우선 권장 — 모바일 URL 이 분리되어 있고 트래픽 큼)
- 화면 사이즈 토큰: 모바일이면 375, 데스크탑이면 1920 (`kds-rules/spec-and-bridge.md` 참고)
- 생성일: 2026-05-19
- 수집 페이지: 메인 1개 (mobile + desktop)

## Viewport 패턴 판별

| 관찰 | 결과 |
|---|---|
| 모바일 URL 진입 (`shop.kt.com` → JS redirect) | `/main.do` 로 PC 이동, **모바일 UA 일 땐 `m.shop.kt.com/m/main.do` 로 분기** |
| 모바일·데스크탑 캡쳐 비교 | 명백히 **다른 레이아웃** — 모바일은 세로 1열 + 11개 4×3 바로가기, 데스크탑은 좌측 QUICK MENU 사이드바 + 우측 hero + 가로 11개 원형 바로가기 + 헤더 상단에 노출되는 KT 구독·요고·로그인 위젯 |
| URL 분리 여부 | `m.shop.kt.com` vs `shop.kt.com` 도메인 자체가 다름 |

**결론: 적응형 (별도 모바일 URL).** 두 viewport 모두 매트릭스로 보지 말고 **택일** 필요. 콘텐츠 구성은 거의 동일 (같은 빅배너·바로가기 11종·인기상품·핫이슈·기획전·꿀팁·고객센터·당첨자 발표 순서) 이나 모바일은 카드/리스트 위주, 데스크탑은 가로 그리드/사이드바 위주.

## 데스크탑 차이점

- 좌측 사이드 **QUICK MENU** — `로그인 없이 간편하게` + 빠른 기기변경 / 갤럭시 / 인기기획전 등 카테고리 + `인기기획` 헤딩 + 핸드폰 / 인터넷/TV / 요고 / 액세서리 / 가전 등 보조 nav
- 상단에 상시 노출되는 상단 위젯: `요고` 배지 + `KT 구독` 배지 + 사람 / 검색 / 장바구니 / 마이 아이콘 + `회원가입 선물받기` 배너
- 헤더 GNB: `Shop` (현재 선택) + `상품` / `혜택` / `고객지원` / `마이`
- 빅배너 hero 가 **좌측 텍스트 + 우측 일러스트** 좌우 분할 (모바일은 단일 이미지)
- 바로가기 11개가 **가로 1열** 원형 칩 + 텍스트 (모바일은 4×3 그리드)
- 인기상품이 **3-col 그리드** + 토글 (월할부금 펼침)
- 우측 하단 고정 플로팅: "최근 본 상품" 카드 + 상담 플로팅 (채팅·카톡·전화)
- 좌측 카테고리에 **"7% 요금할인"** 빨강 배지가 강조됨 (KT Shop 의 핵심 차별 메시지)

## UX 라이팅 톤 관찰

- **문체**: 해요체 + 반말 혼재. **`해요체`가 베이스**, 액션 유도엔 평어/명령어 ("지금 iPhone 17로 바꿔야 할 이유!", "마음 속에만 있던 진심, 지금 전해보세요!").
- **이모티콘·기호**: `♥`, `!` 다수 사용. 광고/배너 카피에서 강한 감정 톤.
- **숫자 강조**: 가격·할인율은 별도 `<em>` 으로 강조. "월 51,150원" 식. "7%" / "32%" / "65%" 등 퍼센트는 큰 폰트.
- **할인 표현 패턴**: "**최대 X% 할인**" / "**월 X원**" / "**X만원 혜택**" / "**할부원금 0원**" / "**총 할인액**" — 가격 임팩트 직접적
- **CTA 패턴**: "바로가기" / "자세히보기" / "전체보기" / "기획전이 궁금해요" / "더보기" — 친근·직접적
- **카피 호흡**: 짧은 문장 + ! 마침 (예: "0원폰 라인업 / 액세서리 4종 패키지까지! / 0원")
- **이벤트 캐치프레이즈**: "오직 KT닷컴에서만!" / "통신사 유일!" / "KT닷컴 고객만을 위한 혜택!" — 차별·소속감 강조
- **고객센터 멘트**: 정중체 ("일요일 및 공휴일(대체 휴무일 포함)은 온라인 상담만 가능") — 안내성 텍스트는 격식체로 분리
- **꿀팁 영상 자막**: 친근한 페르소나 ("나과장이 알려주는 통신-꿀팁") — 캐릭터 마케팅 활용

- **섹션 순서 (반드시 이 순서로)**: 상단바 → 카테고리탭 → 빅배너 → 바로가기 그리드 → 핫딜 배너 → 인기상품(nested tab) → 유선기획전 슬라이드 → Hot Issue → 기획전 → 꿀팁 → 고객센터 → 한줄공지/당첨자발표 → 하단 nav
- **원본 문구 보존 항목**: 빅배너 카피, 상품명, 가격 (월 단위 금액·할인액 모두), CTA 라벨 ("바로가기" / "자세히보기" 등), 고객센터 전화번호 + 운영시간, "오직 KT닷컴에서만" / "통신사 유일" 류 차별 메시지
- **주의사항**:
  - 컬러 매핑: KT Red (`#d71826` 등) → **KDS `red.500` (가장 가까운 primitive)** 으로 치환 (직접 hex 금지). KT Teal (`#01a69f` / `#01a2ba` 등) → **KDS `teal.500` 또는 `teal.300`** 로 치환
  - shop.kt.com 은 **dual accent (Red + Teal)** 구조이므로 시안의 Accent Primary 선택에서 **Red / Teal 중 어느 것을 Accent Primary 로 둘지** 가 차별화 축이 됨. brand.md 의 컬러 시그니처 결론을 따른다 (포지셔닝상 Red 가 메인이지만, "요고" 강조 시안에서는 Teal 도 Accent Primary 후보)
  - 실제 이미지는 `/preview-assets/shop-kt-main/...` placeholder 로 사용 (또는 KDS placeholder)
  - 원본 가격 텍스트의 강조 `<em>` 은 KDS typography `heading-2` 또는 `title-1` 로 매핑
  - 빨강 할인 배지 ("7% 추가할인" 등) 는 시안의 Accent Primary 가 Red 면 그대로, Teal 이면 별도 Accent Secondary (red.500) 로 분리
