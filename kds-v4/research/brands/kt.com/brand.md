# KT (kt.com) — Brand Sheet

**관찰된 영역**: shop.kt.com 메인 (2026-05-21)
**brand_mode**: extend
**도메인 통합**: kt.com (서브도메인: shop / login / m.shop / corp 등 — 동일 브랜드)

## 1. Voice / Tone

- 핵심 단어: **신뢰**, **빠르다**, **혜택**, **선택권**
- 호칭: "고객님" (격식) — 단 마케팅 헤드라인은 짧고 직설적 ("지금 바꾸세요", "이번 달만")
- 마이크로 카피: 한 줄 카피 + 강조 키워드(가격/할인/한정) 강조 — 본문은 정보 위주
- 톤 폭: 차분한 안내(서비스 영역) ↔ 강한 프로모션(쇼핑 영역). 같은 사이트 내 양극단 공존

## 2. UX 모티프

- **단계형 컨버전** 강함 — 단말 선택 → 요금제 선택 → 결합·옵션 → 신청. 화면을 가로지르는 progress 가 자주 노출됨
- **혜택 가시화** — 할인가/지원금/제휴 카드를 카드 정면에 박는 패턴
- **카테고리 ↔ 상품 카드** 교차 — 카테고리 진입 후 가로 스크롤 상품 캐러셀 + 세로 카드 그리드 혼합
- **검색 우선** — 헤더 검색바가 항상 상단에 노출. 자급제/단말명 직접 검색 흐름

## 3. 컬러 시그니처 (관찰)

- **KT Red `#E20613`** — 핵심 CTA, 가격 강조, 이벤트 뱃지. 절제된 빈도 (헤더·CTA·강조 키워드만)
- **검정 / 다크 그레이** — 텍스트 1차, 헤더 배경(이벤트성)
- **흰색** — 기본 surface
- **회색 단계** — 카드 구분선, 비활성 영역

→ **KDS 매핑**: `fill.accent-primary = red.500 (#e0282f)` 그대로 사용 (KDS 시맨틱 기본값). KT 의 실제 `#E20613` 은 KDS primitive 토큰 외 hex 이므로 가장 가까운 `red.500` 으로 매핑. 시안 전반에서 **Accent Primary 1개 자리**는 red 로 통일

## 4. 비주얼 성향

- **사진 사용 비중 큼** — 상품 사진 + 인물 모델 사진 + 라이프스타일 컷. 모바일에서도 큰 hero 이미지 노출
- **그리드 성향**: 명확한 카드 (4~6px ~ 12px corner radius), 충분한 padding, 카드 사이 명확한 구분
- **아이콘**: line + filled 혼합. 카테고리 아이콘은 line 류, 강조 영역은 filled
- **배경**: 거의 항상 white surface. 다크 헤더/배너는 제한적 사용 (프로모션 강조 시)

## 5. 타이포 성향

- 한글 본문: 산세리프 ("KT산스" 류) — KDS 에서는 Pretendard / KDS 표준 typography 토큰으로 매핑
- 위계: 큰 헤드라인(28~32) + 강조 키워드 가격(20~24, weight 700) + 본문 14~16
- letter-spacing: 약간 좁힘 (`-0.3 ~ -0.5px`) — KDS typography semantic 그대로

## 6. 포지셔닝

- "이동통신·디바이스 1위 사업자" — 신규 단말 출시 1day 단독 혜택, 결합 혜택 등 **'가장 빠르다 / 가장 많다 / 가장 안전하다'** 의 종합 강조
- 경쟁자(SKT/LGU+) 대비 차별화: **요금제 + 단말 + 인터넷·TV 결합** 의 종합 묶음 컨버전
- 쇼핑 영역(shop.kt.com) 은 본사 서비스 영역과 분리된 **e커머스 톤** — 더 강한 프로모션, 더 직접적인 가격 노출

## 시안 변형 후보 풀 (이 brand 에서 자동 도출)

- **시안 B (톤)**: brand 의 "Red Accent + 차분한 본문" 양극을 강도 다르게 표현 → 후보: (B1) red 강한 강조 + 큰 타이포, (B2) red 절제 + 흑백 위주, (B3) red + 다크 헤더 hero
- **시안 A (레이아웃)**: brand 의 "큰 hero + 카드 그리드" 모티프를 레이아웃으로 흔듦 → 후보: (A1) 좌우 분할 hero + 카테고리 그리드, (A2) 풀폭 hero + 가로 캐러셀, (A3) 카드 적층형
- **시안 C (UX)**: brand 의 "단계형 + 검색" 양극 → 후보: (C1) 검색 + 단계 indicator 우선, (C2) 카테고리 탭 + 큐레이션, (C3) 단말 비교 우선
