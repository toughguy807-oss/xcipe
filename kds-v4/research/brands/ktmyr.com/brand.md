## 6가지 결론

### 1. Voice (서비스 고유의 일관된 개성)

밝고 친근하면서도 통신 서비스의 신뢰감은 유지. **"~해주세요 / ~해보세요" 의 해요체** 일관 — 명령형(하십시오체) 아님, 반말도 아님. 친구처럼 권하는 톤 ("로그인하고 내 알뜰폰 정보, 쉽게 확인해보세요", "더 편리하게 이용해보세요", "두 번 클릭으로 내 요금제를 찾아보세요!"). 광고문구 빈도 높음(느낌표 자주).

> **셀프개통 영역 보강 (2026-05-18)**: 같은 해요체 패턴 일관 유지 — "~확인하실 수 있어요!", "~안내 드려요.", "5분 만에 셀프개통 하기" (시간 약속 + 행동). 가치제안 문구 ("내 정보를 한눈에!", "누구나 쉽게 개통!") 의 느낌표 사용도 메인 홈과 동일 강도. **충돌 없음** — Voice 결론 그대로.

> **후불요금제 목록 영역 보강 (2026-05-22)**: 정보 페이지 특성상 권유체 빈도 ↓, **명사형 종결 ↑** — 카테고리 칩 ("인기 요금제", "제휴혜택", "부모님"), 정렬 옵션 ("추천 순", "데이터 많은 순"), 가격 보조 ("평생 할인", "7개월 이후 4,800원"). 단 검색 placeholder "검색어를 입력해 주세요." 와 CTA "비교하기" 는 해요체·동사형 유지. **충돌 없음** — Voice 결론 그대로, 영역 기능 (정보 노출) 에 맞춰 권유 빈도만 조절.

### 3. UX 모티프 (사이트 전반에서 반복되는 흐름 패턴)

- **양축 컨버전 강조**:
  - **즉시 진입 (로그인)** = 기존 회원
  - **즉시 가입 (회원가입 — 퍼플 강조)** = 신규 고객
  - 두 CTA 를 같은 비중·인접 위치에서 동시 노출하여 사용자 분기 부담을 줄임 (로그인 페이지의 [로그인][회원가입] 인접 배치, 메인 홈의 [로그인][회원가입] 인접 배치 — 둘 다 같은 UX 패턴)
- **단계형 큐레이션 UX**: "내 요금제 찾기" (데이터 → 통화 → 결과 추천) 같은 챗봇 형식 단계 진행. (메인 영역 관찰) **+ 셀프개통 영역에서 동일 패턴 확장 관찰** (2026-05-18): 6 step 가이드 (사전 준비 → 요금제 → 본인 인증 → 신규/번호이동 → 유심 정보 → 납부) — segmented step indicator + active 상세 패널의 큐레이션 흐름. 메인 홈의 단계형 UX 모티프가 콘텐츠형 페이지에서도 일관 적용됨. **brand 의 핵심 UX 시그니처**.
- **앱 진입 영구 노출**: 모바일에서 화면 하단에 "앱 다운로드 권유 카드" 가 고정 노출. dismiss 문구도 친근체. 웹 사용자도 앱 진입으로 유도하는 강한 의도. (셀프개통 영역에서는 챗봇 FAB 가 유사 역할 — 페이지 어디서나 진입 가능한 보조 채널)
- **소셜 로그인은 보조**: 네이버 로그인을 ID/PW 본문 아래 "다른 방법으로 로그인" 구분선 뒤에 배치. 메인 진입로 아님 — KT 자체 계정이 1순위.

### 5. 비주얼 모티프 (사진 vs 일러스트 비중, 그리드, 카드, 이미지 비율)

- **일러스트 매우 강함 (메인·셀프개통)**. 사진은 거의 없음. 마스코트("앙뜰"-고양이 캐릭터 / 토끼 / 돼지) + 의인화된 사물(휴대폰·선물상자·구름) 가 hero·카드·이벤트마다 등장.
- **카드 라운드 큼** (눈대중 24~32px in 셀프개통, 12~20px in 후불요금제). 라운드된 카드 안에 일러스트 + 큰 헤딩 + 짧은 설명.
- **그리드 약함 (메인·셀프개통)**. 카드 사이에 분명한 격자 정렬보다는 자유로운 여백·이질적 배경 컬러로 섹션 구분.
- **이미지 비율**: 가로 16:9 hero 보다 **사각·세로 카드** 가 더 많음. 캐릭터 일러스트는 카드 한쪽에 떠있는 형태.
- **모바일·데스크탑 모두 캐릭터 일러스트는 유지 (메인·셀프개통)**. 모바일에서도 일러스트 크기 줄이지 않고 충분히 표시.

> **셀프개통 영역 보강 (2026-05-18)**: 비주얼 결론 강화 — 일러스트 사용이 페이지 곳곳에 (Hero 2장 + Step 가이드 6장 + Option 카드 2개) 일관. 의인화 사물 (스쿠터+박스 = 바로배송, 장바구니 = 편의점, 휴대폰·신분증 = 가이드) 모티프가 콘텐츠 페이지에서도 유지. **사진 0장 / 일러스트 다수**. 카드 라운드는 30px (Hero/Info), 20px (Plan), 12px (FAQ) 의 **계층적 라운드** — 큰 컨테이너일수록 더 큰 라운드. 충돌 없음.

> **후불요금제 목록 영역 보강 (2026-05-22)**: **비주얼 모티프가 영역에 따라 분기**:
> - 메인·셀프개통 = 캐릭터 일러스트 + 의인화 사물 (활기·친근감)
> - **후불요금제 = 일러스트 0장** — 캐릭터·의인화 사물 완전 부재. 카드 안 사업자 로고 (외부 brand 자산) 만 시각 요소. 그리드는 **단일 컬럼 균등 카드** (모바일·데스크탑 모두) — 250개를 균질 비교 가능하도록 격자 정렬 강함.
> - 이모지 아이콘 (`👍 인기 요금제`, `🎁 제휴혜택`, `👵 부모님`, `🧑 청소년`, `♾️ 완전 무제한`) 으로 카테고리 칩 시각화 — 일러스트 대체 수단. brand 의 친근 톤은 이모지로만 유지.
> - 카드 그림자 옅음 (눈대중 `0 2px 8px rgba(0,0,0,0.04~0.06)`), 라운드 12~20px — 셀프개통보다 절제. 정보 페이지 특성.
> - **이 차이는 brand 결론과 충돌 아님 — 영역별 의도적 분기**. brand 의 "활기 + 신뢰" 양면 중 정보 페이지는 "신뢰" 면에 가깝게 변형.

### 6. 타이포 위계 성향 (폰트는 KDS = KT Flow + Pretendard 고정, 위계만 다룸)

- **헤딩 큰 강도**. 메인 h2/h3 가 42px / **weight 300** — **lighter weight 의 큰 헤딩** 이 특징. 묵직한 검정 large heading 이 아니라 가볍게 큰 표제로 brand 의 친근 톤 표현.
- **로그인 페이지 h1 "로그인"**: 모바일 26px weight 700 / 데스크탑 32px weight 700 — 폼 안 헤딩은 더 굵게. 즉 **메인은 light heading, 폼은 bold heading** 으로 영역별 weight 다름.
- 본문 14~16px regular. 보조 문구 12~14px.
- **정렬은 좌측 정렬** 일관. 중앙정렬은 hero 캐러셀 인디케이터 등 좁은 영역에만.
- 텍스트 색은 검정(`#000000`/`#333333`) 또는 회색(`#555555`/`#888888`). 컬러 텍스트는 거의 없음 — 컬러는 배경·아이콘 위주.

> **셀프개통 영역 보강 (2026-05-18)**: 영역별 weight 차이 패턴 재확인 — Hero H3 (캐러셀) 가 20~26px / **weight 400** (lighter heading), Section H2 (`셀프개통 가능시간` 등) 가 18~22px / weight 700 (bold), step 숫자 14~16px / weight 700. **메인 홈의 lighter heading 보다는 가벼움 약하지만 (셀프개통은 정보 페이지라 콘텐츠 가독성 우선), 캐러셀 hero 만은 lighter weight 의 brand 특징 유지**. 가격 위계는 사이즈로만 (정가 14px 취소선 → 할인가 26~32px weight 700) — **컬러 강조 없는 절제 톤** 이 brand 특징. 영역별 차이 메모: 셀프개통은 **메인 홈 대비 헤딩 weight 가 다소 무거움** (콘텐츠 가독성 우선). 충돌 아닌 미세 차이.

> **후불요금제 목록 영역 보강 (2026-05-22)**: **lighter heading 패턴 완전 부재** — 페이지 헤딩 "후불요금제" h2 16px / **weight 700**, 카드 가격 17~22px / **weight 700**, 카드 이름 14~18px / **weight 700**. 모든 강조가 bold. 메인의 light large heading 패턴은 이 영역에서는 적용되지 않음 — 정보 페이지로서 가독성·정보 강조 우선. **단 가격 위계는 셀프개통과 동일 절제 패턴**: 정가(`7개월 이후 4,800원`)는 회색 작은 보조, 할인가(`월 900원`)는 큰 강조. 다만 셀프개통과 다르게 **컬러로도 강조** (cyan-teal `#14BAD7`) — 가격에 컬러 강조를 처음 도입한 영역. 영역별 차이: 셀프개통=사이즈만 / 후불요금제=사이즈+컬러. 충돌 아닌 영역 특성. 정렬 옵션 active = weight 400 (강조 약함, 비활성은 weight 300) — KDS `body-2 / body-3` 매핑.
