# Brand: shop.kt.com (KT Shop / KT 닷컴)

> 도메인 단위 브랜드 정보. 같은 도메인 (`shop.kt.com` + 적응형 모바일 URL `m.shop.kt.com`) 의 모든 화면 작업이 공유.
> KT 본사이트 (`www.kt.com`) 와는 같은 KT 그룹이지만 **톤·컬러 운용이 분리**됨. 본사이트는 회색·블루 톤 기업 사이트, shop.kt.com 은 빨강·민트 강조의 e커머스. 이 brand.md 는 **shop.kt.com 전용** (kt.com 본사이트는 별도 도메인 폴더 필요 시 분리).

## 포지셔닝 한 문장

**빨강·민트 dual accent 의 활기차고 직접적인 통신 이커머스. "할인·혜택을 가장 명확하게 보여주는 곳" — 가격 강조와 차별 메시지("오직 KT닷컴에서만") 가 핵심 톤. 정보 밀도 높은 카드·캐러셀 구조로 다양한 프로모션을 일관 수용.**

## 1. Voice (서비스 고유의 일관된 개성)

**친근·활기·직접적**. 격식보다는 **혜택을 빠르게 알리는 영업·캠페인 톤**.

- 친근함: "요고가 추천해드려요", "쉽게보는 꿀팁!", "나과장이 알려주는 통신꿀팁" 같은 의인화·캐릭터 마케팅 (라온 고양이, 나과장 페르소나)
- 활기: `!` 마침 빈도 매우 높음, 이모티콘 (`♥`) 등장, 짧은 호흡 문장
- 직접적: "지금 iPhone 17로 바꿔야 할 이유!" 처럼 행동 유도가 우회 없이 본론으로
- 단 안내·법적 영역 (고객센터 운영시간 안내, 비밀번호 정책 등) 은 격식체 분리 — 영역별 분기 명확

KT 본사이트 (`kt.com`) 의 차분·기업적 톤과는 구분됨. shop.kt.com 은 **이커머스 전환 톤**.

## 3. UX 모티프

**큐레이션형 + 캐러셀 다중 적층 + 카드 그리드 우선**.

- **큐레이션형**: 빅배너 (최대 20장 자동 롤링), Hot Issue (3장 캐러셀), 기획전 (4장 가로 스와이프), 꿀팁 동영상 (다수) — **각 영역이 모두 캐러셀** 구조. 사용자가 직접 검색·필터하기 전에 **큐레이션된 카드를 스와이프로 보는** 흐름이 베이스
- **카드 그리드 우선**: 바로가기 4×3, 인기상품 2×2, 유선기획전 가로 스와이프, 기획전 가로 스와이프 — 정보를 **카드로 적층**해서 위에서 아래로 훑게 함
- **단계형 보조 흐름**: "취저 핸드폰 빨리 찾기" 9단계 step 모달 — 핵심 흐름은 큐레이션이고, 단계형은 **보조 도구**로만 존재
- **검색은 보조**: 헤더에 검색 아이콘 있지만 메인에서 강조 안 됨. 사용자가 "이 카드를 보고 들어가는" 흐름이 압도적
- **세션 (장바구니·최근 본 상품) 활용**: PC 에 "최근 본 상품" 고정 플로팅, "장바구니" 헤더 아이콘 상시 노출 — 이커머스의 표준이지만 메인에서는 강조 약함

## 4. 컬러 시그니처 의도

**KT Red + KT Teal 의 dual accent. 두 컬러가 동등 강도로 등장하나 역할 분리 명확.**

- **KT Red** (`#d71826` 외): **할인·혜택의 강도 자체**를 시각화. 배지·가격·할인%·"7% 할인"·강조 CTA. 사용자가 화면을 훑을 때 가장 먼저 끌리는 시각 무게 중심
- **KT Teal** (`#01a69f` 외): **"요고" 서비스의 시그니처** + 인터넷·다이렉트할인 카테고리의 강조. 캐러셀 페이저·active 상태·인터넷 디자인 디바이스 영역
- **두 컬러의 동등성**: 어느 한쪽이 메인이고 다른쪽이 보조라기 보다는, **카테고리·메시지 성격에 따라 두 컬러가 갈라쓰임**. 예: 같은 바로가기 11개 카드에서 "혜택성 카테고리 (핸드폰구매·갤럭시·Apple·인터넷 가입·요고·기변)" 는 teal pastel (`#e8f8f6`) 배경, "기능성 카테고리 (액세서리·USIM·핫딜·캐시리워드·마이샵)" 는 grey-blue pastel (`#f1f5f8`) 배경 — 이 구분 자체가 brand 의 디자인 의사결정
- **회색 위계 + 검정 + 빨강 + 민트** 의 4-color palette 가 KDS Brand Color Case 1/2 의 "회색 5단계 + 검정 + Accent1 + Accent2" 구조와 정확히 맞물림

### 시안 적용 시 KDS 매핑 권고

- **시안 B (톤 우선) 의 Accent Primary 후보**:
  - 후보 1: **Red Primary + Teal Secondary** (메인 화면의 화면 시각 무게중심 그대로) — KDS 토큰: Accent Primary = `red.500`, Accent Secondary = `teal.500`
  - 후보 2: **Teal Primary + Red Secondary** ("요고" 차별화 메시지를 더 부각) — KDS 토큰: Accent Primary = `teal.500`, Accent Secondary = `red.500`
  - 후보 3: 회색·검정 위주 모노톤 강조 + 단일 Accent (Red 또는 Teal 한쪽만) — 정보 위계 강화 방향
- **임의 hex 절대 금지**. 원본의 `#d71826` 은 `red.500` 으로, `#01a69f` 는 `teal.500` 으로 치환

## 5. 비주얼 모티프

**사진 비중 압도적 + 일러스트 보조 + 그리드 강도 강함 + 카드 사용 매우 빈번 + 4:3·1:1 비율 혼재**.

- **사진/상품 컷**: 빅배너 (실사·합성)·상품 카드 (흰 배경 실사 컷)·핫딜 (이미지)·꿀팁 (영상 썸네일) — **사이트의 70% 이상이 비트맵 이미지로 채워짐**. 빈 공간보다 시각 자극이 항상 우선
- **일러스트**: 데스크탑 hero 우측 가구 일러스트 (Apple·Galaxy·BTS 캐릭터 등), 라온 고양이 캐릭터 — 보조적 활용. 모바일에서는 일러스트 비중 낮음
- **그리드 강도**: 바로가기 4×3, 상품 카드 2×2/3-col, 기획전 4-col, 고객센터 메뉴 4×2 — **균일 그리드의 적층**이 패턴
- **카드 사용 빈도**: 거의 모든 콘텐츠가 카드. 단순 리스트나 텍스트 블록은 거의 안 보임 (한줄공지·당첨자발표만 단순 리스트)
- **이미지 비율**: 빅배너 16:9 / 상품 카드 1:1 / 기획전 가로 카드 1.6:1 (4:3 에 가까움) — 혼재됨, 영역마다 비율이 다름
- **흰 배경 비중 높음**: 카드 사이 사이 흰 배경이 명확. 검정 dark surface 거의 없음 (상품 카드 이미지 자체만 검정 배경)

## 6. 타이포 위계 성향

**큰 가격 강조 + 굵은 weight 다용 + 짧은 한글 카피 + 영문 헤딩 혼재**.

- **헤딩 크기 강도 강함**: "인기상품" / "기획전" / "고객센터" 등 섹션 헤딩이 충분히 크고 굵음 (h2 사용)
- **weight 사용 빈도 매우 높음**: 가격 `<em>`, 강조 텍스트, 배지, 헤딩 모두 굵은 weight. **광고/배너 카피의 70% 이상이 굵은 weight**
- **숫자 강조**: 가격·할인율 숫자가 큰 폰트 + 굵은 weight + 별도 컬러 (빨강) — 정보 위계상 가장 두드러짐
- **정렬 성향**: **좌측 정렬 위주**. 중앙 정렬은 카드 내부 배지·CTA 에 한정
- **letter-spacing 좁힘**: 한글 카피의 자간을 음수 (`-0.5vw` ~ `-0.7px`) 로 좁혀 시각 밀도 높임
- **영문 헤딩 혼재**: "Hot Issue" / "Apple" / "Galaxy" — 영문은 영문 그대로, 한글은 한글로
- **폰트 자체**: KDS 가 KT Flow + Pretendard 으로 고정이라 다루지 않음 (원본도 별도 KT 전용 폰트 사용)

- 후보 3: **좌측 카테고리 + 우측 콘텐츠 분할** (데스크탑 quick menu 패턴을 모바일에 변형 적용)

### 시안 B (톤 변형) 후보
- 후보 1: **Red Primary + Teal Secondary** (원본 톤 — 빨강 강조 우위)
- 후보 2: **Teal Primary + Red Secondary** (요고 차별화 우위)
- 후보 3: **모노톤 + 단일 Accent** (정보 위계 강화 — 회색·검정 베이스 + Red 또는 Teal 한쪽만)

### 시안 C (UX 변형) 후보
- 후보 1: **큐레이션형** (원본 — 빅배너 + 카드 적층 + 캐러셀 다중)

- 후보 3: **단계형 가입 유도** ("취저 핸드폰 빨리 찾기" 같은 step 도우미를 메인의 핵심 동선으로 끌어올림)
