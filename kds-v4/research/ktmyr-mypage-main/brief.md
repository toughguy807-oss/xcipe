## UX 라이팅 톤 (관찰)

- "**간편하게 가입, 개통, 관리하세요!**" — 명령형이 아닌 권유, 행동 3개 병렬 + 느낌표
- "**로그인하고 내 알뜰폰 정보, 쉽게 확인해보세요.**" — 해요체, 친구 권유
- "**원하는 요금제가 없으신가요? 두 번 클릭으로 내 요금제를 찾아보세요!**" — 질문형 + 시간/단계 약속
- "**한 달에 데이터를 얼마나 사용하시나요?**" — 큐레이션 단계 안내 질문
- 카테고리 라벨: `완전 무제한 / 제휴혜택 / 부모님 맞춤 / 청소년 맞춤` — 명사형
- 가격 표기: `월 4,800원` (정가, 회색 취소선) → `월900원` (할인가, 큰 컬러 강조)
- 셀프개통 STEP 라벨: `STEP 01 / STEP 02 / STEP 03 / STEP 04` — 라벨 위 + 짧은 설명 아래

## 일러스트·이미지 대체 정책

원본 사이트는 캐릭터 마스코트(앙뜰·토끼·돼지) + 의인화 사물(휴대폰·선물상자·구름) 적극 사용. 본 시안은 브랜드 캐릭터 자산 없으므로 다음으로 대체:
- **그라디언트 배경** (Purple/Teal/Pink/Lime 의 KDS primitive 매핑) — hero 카드, Quick Action 카드
- **KDS svg 아이콘 라이브러리** (gift, smartphone, qr-code, percent, heart, calendar, document) — Quick Action, 부가서비스 카테고리
- **이모지** — 카테고리 칩 (👍 인기 / 🎁 제휴혜택 / 👵 부모님 / 🧑 청소년 / ♾️ 완전 무제한)
- **그라디언트 + 큰 텍스트** — 캐릭터 자리 (예: hero 카드의 큰 강조 텍스트)

## brand.md 적용 결정 (디자이너 자동 채택)

- Accent Primary: **purple.500 (`#6941ff`)** — brand.md 권장 + 회원가입 CTA 우세
- 톤 변형 시안 B: Accent Primary 를 **teal.500 (`#007f7f`)** 로 — 헤더 로고 톤 carry-over
- Voice: 해요체 일관, 권유체 ("~해보세요"), 느낌표 적당히 사용
- 카드 라운드: **16~24px** (카드 컴포넌트는 8/12/16/20/24 KDS radius 토큰 안)
- 헤딩: 메인 섹션 헤딩 = lighter weight (KDS `heading-3 / title-1`) + 큰 사이즈 (brand 의 "큰 lighter heading" 특징 carry)
- 가격: 정가 작은 취소선 회색 → 할인가 큰 weight 700 (시안 A·C 는 Purple, 시안 B 는 Teal)
- 빈도: Purple/Teal 자리는 한 시안 안에 1개 일관 유지. Accent Secondary 는 별개 (예: 시안 B 의 Accent Secondary 로 Purple 약 강조 가능)
