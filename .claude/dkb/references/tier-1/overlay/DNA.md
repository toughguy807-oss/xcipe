## 정확 토큰 (실측, 2026-05-06)

### 컬러
- Background: **`#FBF9FB`** (`--white--bg`) — lavender-tinted white
- 텍스트 primary: **`#2E2F53`** (dark blue) — body color
- blue 4단계: 20% `#2E2F5333` / 40% `#2E2F5366` / 60% `#2E2F5399` / 100% `#2E2F53`
- gray 400: `#B3B3B3`
- button hover: `#2E2F53`

### 폰트
- body: **`Roboto Flex`** (Variable Sans) — Modern Web Font
- H1/H2: **`Ppeditorialold, Arial, sans-serif`** (Pangram Pangram Editorial Serif)
- H1: **72px / weight 400 / letter-spacing -2.16px / line-height 75.6px**
- H2: 72px / weight 400 / -2.16px (동일)
- H3: 2.25rem (`--typography--h3`)
- H4: 1.125rem (`--typography--h4`)
- typography line-height: H6 0.975rem / p-s 1.28rem

### 그리드 / spacing
- **156 vars (Batch 5 최다)** — Webflow Custom Code 시그니처
- paddings 11단계: padding-1~11 (1.5rem ~ 5rem)
- spacing px-rem 매핑: 20px-1.25rem / 28px-1.75rem / 32px-2rem / 84px-5.25rem / 92px-5.75rem / 96px-6rem / 116px-7.25rem / 148px-9.25rem / 200px-12.5rem / 216px-13.5rem / 288px-18rem / 352px-22rem / 416px-26rem
