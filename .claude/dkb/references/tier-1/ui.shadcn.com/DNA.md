## 정확 토큰 (Vision 추정)

### 컬러
- Background: `#FFFFFF`
- Background card: `#FAFAFA` 추정
- Text primary: `#0A0A0A` 또는 `#09090B` (Pure black 회피)
- Text secondary: `#71717A` (zinc-500)
- Border subtle: `#E4E4E7` (zinc-200)
- Accent CTA: 검정 알약

### 폰트
- Sans: Geist 또는 Inter / weight 400-700
- Mono: Geist Mono / weight 400-500

### 컴포넌트 시그니처
- Card / Form / Toggle / Switch / Dropdown / Popover / Toast / Calendar
- Charts (Bar / Line / Area / Pie)
- Sidebar / Navigation Menu
- Dialog / Alert Dialog / Sheet
