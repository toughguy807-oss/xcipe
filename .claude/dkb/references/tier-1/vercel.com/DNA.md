## 정확 토큰 (추정, Vision 검증 필요)

### 컬러
- Background: `#000000` (Pure black 사용 — Vercel은 예외적)
- Background alt: `#0A0A0A`
- Background card: `#111111`
- Text primary: `#FFFFFF` (Pure white 사용)
- Text secondary: `#A1A1A1`
- Accent cyan: `#00DFD8` 추정 (네온 시안)
- Accent magenta: `#FF0080` 추정 (네온 마젠타)
- Accent gradient: linear-gradient(120deg, #00DFD8, #FF0080)

### 폰트
- Display: Geist Sans / weight 400-700
- Body: Geist Sans / weight 400 / size 14-16px
- Mono: Geist Mono

### 간격
- Section 간격: ~120px
- Container max-width: 1280px
- Bento gap: 16-24px

### 모션
- Scroll-triggered fade + chart animation
- Hover: ~150ms + 네온 글로우
- Cursor-aware effect (마우스 따라가는 빛)

2. **Bento Grid 5종 변주** — Web Apps / Agents / AI Apps / Composable Commerce / Multi-tenant Platform
