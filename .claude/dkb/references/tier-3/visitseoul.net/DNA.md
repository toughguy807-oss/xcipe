## 정확 토큰 (Vision 추정)

### 컬러
- Background hero: 사진 풀블리드 (벚꽃 핑크/그린)
- Background widget: `rgba(255,255,255,0.9)` 반투명
- Text primary: `#1A1815`
- Text on photo: `#FFFFFF`
- Accent: 거의 부재 (사진 톤 의존)

### 폰트
- Korean: Pretendard 추정
- English: 자체 sans
- 다국어 폰트 9종 자동 전환

### 시그니처

1. **4계절 사진 풀블리드 Hero** — 벚꽃/여름/단풍/눈 자동 전환
2. **위젯 카드 (실시간 날씨/온도)** — 정량 정보 + 친근감
3. **9개 언어 토글** — 한/영/일/중간/중번/러/말/스/Partners (다국어 톱)

## 적용 가이드 (한국 공공/관광 직격)

### 적용 가능 영역
- **공공/관광 Hero**: 자연 풀블리드 사진 + 위젯 카드
- **다국어 처리**: 한국어 메인 + 다국어 자동 전환 + 폰트 fallback
- **계절 시그니처**: 4계절 nav + 사진 자동 전환
- **앱 install QR**: iOS/Android QR + 위젯 결합

### 적용 부적합
- ❌ B2B 권위 / Industrial AI (친근 공공 톤 부적합) → Linear/Augury
- ❌ Pure dark 권위 → Scale/Palantir 권고

### 가중치 추천 (한국 공공/관광)
- 1순위 적용: 자연 풀블리드 사진 / 다국어 9종 / 4계절 nav
- 2순위 적용: 위젯 카드 (실시간 정보) / 앱 install QR
- **이건 한국 공공/관광 시각 시그니처 톱 — visit{지자체} 류 직격**
