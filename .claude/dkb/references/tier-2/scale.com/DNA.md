## 정확 토큰 (Vision + 추정)

### 컬러
- Background: `#000000` (Pure black — Scale은 예외적)
- Background alt: `#0A0A0A` 추정
- Text primary: `#FFFFFF`
- Text secondary: `#A1A1A1` 추정
- Accent gradient illust: linear-gradient(135deg, #F4D8E8 0%, #C5E4F0 100%) — 파스텔 핑크 → 파스텔 시안
- Border subtle: `#1F1F1F` 추정

### 폰트
- H1: sans-serif (Inter-like 또는 자체 폰트) / weight 300-400 (light) / size clamp(2.5rem, 5vw, 4.5rem) / letter-spacing -0.02em
- Body: sans-serif / weight 400 / size 14-16px
- Mono: 미확인

### 간격
- Section 간격: ~120px 추정
- Container max-width: ~1280px
- Hero 여백: 좌측 ~100px

### 모션
- Scroll-triggered fade 추정
- Hover transition: ~150ms 추정
- 3D 일러스트 정적 (parallax 미관측)

## 시그니처 요소 (★)

1. **3D 핑크/시안 파스텔 그라디언트 일러스트** — 삼각형 화살표 형태 (Scale 로고 모티프) + 그림자 — Frontier AI 시그니처
2. **그라디언트 텍스트 H1** — light weight + 시안 옅은 그라디언트 적용
3. **Pure black + light weight + 미니멀 nav** — Frontier AI 권위 표현
