## 정확 토큰 (Vision + 추정)

### 컬러
- Background: 다크 사진 풀블리드 (`#000000` 오버레이)
- Text primary: `#FFFFFF`
- Border subtle: 투명 / 흰색 outlined
- Accent: 거의 부재 (절제 시그니처)

### 폰트
- H1: sans-serif (NAVER 자체 또는 Pretendard) / weight 600-700 / size clamp(2.5rem, 5vw, 4rem) / letter-spacing 0.05em (대문자)
- Body: sans-serif / weight 400 / size 14-16px

### 모션
- Carousel slide 자동 전환
- Arrow hover
