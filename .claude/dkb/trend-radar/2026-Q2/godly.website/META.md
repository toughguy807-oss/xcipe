## 사용 방식

### 패턴 1: AI 도메인 시그니처 추출
```
방문: https://godly.website/?category=ai
카드 thumbnail로 AI 도메인 톱 시그니처 한눈 확인 (Hero 그라디언트 / Bento / Italic 등)
```

### 패턴 2: 분기별 Top 큐레이션 추출
```
방문: https://godly.website
주간 / 분기 Top 12 → dkb-analyze 후보 큐 등록
```

### 패턴 3: 트렌드 시그니처 빈도 추적
```
공통 패턴 빈도 (예: 그라디언트 메쉬 8/12, Italic Serif 5/12) → 분기 트렌드 정량
```
