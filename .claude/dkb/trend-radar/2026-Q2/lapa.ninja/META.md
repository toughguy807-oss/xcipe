# lapa.ninja — 메타-레퍼런스

**역할**: "The Best Landing Page Design Inspiration" — 7,300+ 랜딩 페이지 인덱스 (2015~)
**도메인**: 글로벌 랜딩 페이지 톱 / B2B/B2C 통합
**갱신 주기**: 주간 (트렌드 추적 시 분기 1회 추출)
**최초 캡처**: 2026-05-04

## 시그니처

- "Discover more" 카테고리 카드 (Web Design / e-commerce / Inspiration)
- 랜딩 페이지 풀스크린샷 + 비디오 (시각 변화 추적 가능)
- 15,000+ full-page screenshots
- B2B/B2C 통합 카테고리

## 사용 방식

### 패턴 1: 카테고리별 랜딩 페이지 인덱스
```
방문: https://lapa.ninja/category/{cat}/
카테고리: SaaS, Agency, Portfolio, e-commerce, Coming Soon, App, Landing
도메인 매칭된 랜딩 페이지 시그니처 추출
```

### 패턴 2: Hero / CTA / Pricing 섹션 추적
```
방문: https://lapa.ninja/sections/{section}
섹션: Hero, Pricing, Features, Testimonials, Footer, CTA
패턴 카탈로그 (`patterns/{section}/lapa-pattern.md`) 갱신용
```

### 패턴 3: 풀스크린샷 시각 비교
```
랜딩 페이지 1개당 풀 페이지 캡처 보유 → publish-visual-verify Phase 3.6 reference 입력 활용
```
