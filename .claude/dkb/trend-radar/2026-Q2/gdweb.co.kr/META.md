# gdweb.co.kr — 메타-레퍼런스 ⭐ (한국 어워드 톱)

**역할**: 한국 디자인 어워드 (Conversion Design Awards) — 한국 도메인 시그니처 인덱스
**도메인**: 한국 디자인 어워드 / 한국 톤 추적
**갱신 주기**: 분기 1회 (한국 어워드 분기 발표)
**최초 캡처**: 2026-05-04

## 시그니처

- 한국어 메인 사이트 (한글 typography 톱)
- "프로젝트 구축, 에이전시 견적 받기 서비스" — 어워드 + 에이전시 매칭 결합
- 어워드 카테고리: Web / Mobile / Editorial / Public 등
- **한국 컨텍스트 (B2B / 공공 / 관광 / 핀테크) 시그니처 추적 톱**

## 사용 방식

### 패턴 1: 한국 분기별 어워드 추출
```
방문: https://gdweb.co.kr/awards/{year}/
필터: 분기 (1Q/2Q/3Q/4Q)
Top 10 한국 사이트 → dkb-analyze 후보 큐 등록
```

### 패턴 2: 도메인별 카테고리 검색
```
방문: https://gdweb.co.kr/awards/?category={cat}
카테고리: Public(공공), Editorial(매거진), Brand(브랜드), Tourism(관광)
한국 도메인 직격 시그니처 추출
```

### 패턴 3: 한글 typography 톱 추적
```
한국 어워드 입선작은 한글 typography 정밀이 가산 → Pretendard / SUIT / 한글 자체 폰트 사용 사례 추적
```

## 핵심 메타데이터

- **연도별 어워드**: 2024 / 2025 / 2026
- **카테고리**: Web, Mobile, Editorial, Brand, Public, Promotion 등
- **에이전시 매칭**: 견적 의뢰 서비스 (디자인 외주 인덱스)

## 한국 도메인 직격 가치

- 한국 공공/관광/매거진 부족 → gdweb 통해 시드 등재 후보 발굴
- 한글 typography 정밀 사례 추적
- 한국 B2B 시각 시그니처 (toss / 라인 외 후보)
