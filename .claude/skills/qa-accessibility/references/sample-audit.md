## 1. 인식의 용이성 (Perceivable)

### 1.1 대체 텍스트 (1.1.1)

| # | 요소 | 현재 상태 | 판정 | 수정 가이드 |
|---|------|----------|------|------------|
| A-001 | `<img src="hero.jpg">` | alt 미존재 | **FAIL** | `alt="강남구 도심 전경 야경 파노라마"` 추가 |
| A-002 | `<img src="logo.svg" alt="로고">` | alt 불충분 | **WARN** | `alt="비짓강남 로고"` — 브랜드명 포함 |
| A-003 | `<img src="icon-search.svg" alt="">` | 장식 이미지 빈 alt | **PASS** | 기능 아이콘은 `aria-label` 또는 빈 alt 허용 |
| A-004 | `<img src="map.jpg" alt="지도">` | alt 불충분 | **WARN** | `alt="강남구 주요 관광지 위치 지도"` — 정보 전달형 |

### 1.3 콘텐츠 구조 (1.3.1)

```html

```

```
[모달 포커스 트랩]

→ 포커스 트랩 정상 ✓
```
