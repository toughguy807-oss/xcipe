# UI 명세서 (Design-UI) 체크리스트

## UI-ID 체계
- [ ] UI-001부터 연속 번호 부여 (빈번호 없음)
- [ ] 모든 컴포넌트에 고유 ID 존재
- [ ] 유형별 접두어 일관 (UI-BTN, UI-CRD, UI-NAV 등)

## 컴포넌트 완전성
- [ ] GNB/LNB 네비게이션 정의
- [ ] 버튼 (Primary/Secondary/Ghost/Danger)
- [ ] 카드 (기본/이미지/리스트)
- [ ] 폼 요소 (Input/Select/Checkbox/Radio/Textarea)
- [ ] 모달/다이얼로그
- [ ] 테이블/리스트
- [ ] 탭/아코디언
- [ ] 알림/토스트
- [ ] 페이지네이션/브레드크럼
- [ ] FN에서 요구하는 모든 UI가 명세에 존재

## 상태 정의
- [ ] Default / Hover / Active / Focus / Disabled 5상태 전수 정의
- [ ] 각 상태별 시각 변화 명시 (색상, 그림자, 테두리 등)
- [ ] 로딩/스켈레톤 상태 정의 (비동기 컴포넌트)
- [ ] 에러/성공/경고 상태 정의 (폼 요소)
- [ ] Empty State 정의 (리스트, 검색 결과 없음 등)

## STYLE 토큰 참조
- [ ] 색상이 `--color-*` 토큰 참조
- [ ] 폰트가 `--font-size-*` / `--font-weight-*` 토큰 참조
- [ ] 간격이 `--spacing-*` 토큰 참조
- [ ] radius가 `--radius-*` 토큰 참조
- [ ] 하드코딩된 수치 0건

## FN→UI 매핑
- [ ] 모든 FN-###에 대응하는 UI-### 존재
- [ ] orphan UI 0건 (FN 미연결 UI 없음)
- [ ] 추적성 매트릭스 포함 (FR→FN→UI)

## 이미지 소싱
- [ ] 이미지 영역 크기 명시 (width × height)
- [ ] 이미지 비율 정의 (16:9, 4:3, 1:1 등)
- [ ] alt 텍스트 가이드라인 명시
- [ ] 이미지 포맷/용량 가이드 포함

## 접근성 (ARIA)
- [ ] 모든 인터랙티브 요소에 role 정의
- [ ] aria-label / aria-labelledby 명시
- [ ] 키보드 탐색 순서 (tabindex) 정의
- [ ] 포커스 표시자(Focus Indicator) 정의
- [ ] 색상 외 추가 시각 구분 (아이콘, 패턴 등)

## 일관성
- [ ] 동일 기능 컴포넌트 동일 스펙 (페이지 간)
- [ ] 컴포넌트 네이밍 규칙 일관
- [ ] 반응형 3단계(M/T/D) 크기 변화 명시
- [ ] 컴포넌트 간 간격 규칙 통일
