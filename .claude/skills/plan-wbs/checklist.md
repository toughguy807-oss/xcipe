# WBS 품질 체크리스트

> plan-wbs 스킬 실행 후 자체 검증용

---

## 필수 (Must)

- [ ] FN의 모든 기능이 WBS 작업으로 분해되었는가 (FN→WBS 추적성) `[연계 모드 전용]`
- [ ] 각 작업에 담당자가 배정되었는가
- [ ] 각 작업에 기간(일수)이 명시되었는가
- [ ] 선행작업(dependency)이 정의되었는가
- [ ] 마일스톤별 게이트 조건이 명확한가
- [ ] 크리티컬 패스가 식별되었는가
- [ ] 총 기간이 프로젝트 일정 제약 내인가

## 리스크 평가
- [ ] 리스크 최소 5건 이상 식별
- [ ] 점수 6 이상 항목에 대응 전략 + 조치 기재
- [ ] 7대 유형 중 3개 이상 커버

## 공수 산정
- [ ] 업종 보정 계수 적용 여부 확인
- [ ] 대규모(16p+) 시 3점 추정법 사용
- [ ] 버퍼율 명시 (기본 10-15%)

## 권장 (Should)

- [ ] Phase별 작업이 MECE 구조인가 (빠짐/중복 없음)
- [ ] 작업 단위가 1~5일 범위인가 (5일 초과 시 분할 필요)
- [ ] 병렬 수행 가능한 작업이 식별되었는가
- [ ] 리스크 항목별 대응 방안이 구체적인가
- [ ] 산출물 컬럼이 모든 작업에 포함되었는가

## 점검 기준

| 항목 | 기준 |
|------|------|
| FN 커버리지 | FN 항목 100% → WBS 매핑 `[연계 모드 전용]` |
| 작업 크기 | 평균 2~3일, 최대 5일 |
| 의존관계 | 순환 의존 0건 |
| 버퍼 | 전체 일정의 10~15% 여유 |
