# 벤치마킹 평가 기준표

> design-benchmark 스킬 참조용. 100점 정량 채점의 세부 기준과 판단 근거를 정리합니다.

---

## 1. 채점 체계 (100점)

4영역 × 25점 = 100점. 각 영역 5항목 × 5점.

### 점수 등급

| 점수 | 기준 | 의미 |
|------|------|------|
| 5 (탁월) | 업계 상위 5% | 벤치마크 대상으로 적극 참조 |
| 4 (우수) | 업계 상위 20% | 부분 참조 가치 있음 |
| 3 (양호) | 업계 평균 수준 | 기본은 갖춤, 차별화 부족 |
| 2 (기본) | 업계 평균 이하 | 기능은 하나 세련미 부족 |
| 1 (미흡) | 기본 미달 | 반면교사 용도 |

## 2. 영역별 세부 기준

### A. 디자인 (25점)

| # | 항목 | 5점 기준 | 3점 기준 | 1점 기준 |
|---|------|---------|---------|---------|
| A1 | 레이아웃 구조 | 그리드 체계적 + 시각 리듬 완벽 | 그리드 있으나 단조로움 | 그리드 없음, 정렬 불량 |
| A2 | 컬러 활용 | 브랜드 일관 + 대비 AA + 무드 전달 | 컬러 있으나 체계 부족 | 색상 과다/부조화 |
| A3 | 타이포그래피 | 위계 명확 + 가독성 우수 + 서체 조화 | 서체 1~2종, 위계 있음 | 서체 과다, 위계 불명확 |
| A4 | 여백·비율 | 충분한 여백 + 황금비/모듈 적용 | 여백 있으나 불균형 | 여백 부족, 빽빽함 |
| A5 | 시각적 일관성 | 전 페이지 톤·스타일 통일 | 주요 페이지 통일 | 페이지마다 다른 스타일 |

### B. 기능성 (25점)

| # | 항목 | 5점 기준 | 3점 기준 | 1점 기준 |
|---|------|---------|---------|---------|
| B1 | 핵심 기능 완성도 | 업종 필수 기능 100% + 부가 기능 | 필수 기능 갖춤 | 필수 기능 누락 |
| B2 | 검색·필터 | 자동완성 + 필터 조합 + 결과 정렬 | 기본 검색 있음 | 검색 없음 또는 미작동 |
| B3 | 회원·개인화 | 소셜 로그인 + 맞춤 추천 | 기본 회원 기능 | 회원 기능 없음 |
| B4 | 다국어·접근성 | 3개국어+ + WCAG AA 준수 | 2개국어 또는 부분 접근성 | 단일 언어 + 접근성 미비 |
| B5 | 외부 연동 | 지도/SNS/CMS/결제 등 자연스러운 통합 | 1~2개 연동 | 연동 없음 |

### C. 사용성 (25점)

| # | 항목 | 5점 기준 | 3점 기준 | 1점 기준 |
|---|------|---------|---------|---------|
| C1 | 네비게이션 직관성 | 3클릭 내 도달 + 빵부스러기 + 사이트맵 | 메뉴 구조 명확 | 메뉴 혼란, 길 잃기 쉬움 |
| C2 | 반응형 대응 | 3단계(M/T/D) 완벽 + 터치 최적화 | 2단계 대응 | 반응형 미지원 |
| C3 | 로딩 속도 | LCP ≤ 2s + 체감 즉시 | LCP 2~4s | LCP > 4s |
| C4 | 정보 탐색 효율 | 정보 우선순위 명확 + 스캔성 우수 | 정보 있으나 찾기 어려움 | 정보 과부하/혼재 |
| C5 | 모바일 UX | 터치 타겟 48px+ + 스와이프 제스처 | 모바일 사용 가능 | 모바일 사용 불편 |

### D. 혁신성 (25점)

| # | 항목 | 5점 기준 | 3점 기준 | 1점 기준 |
|---|------|---------|---------|---------|
| D1 | 인터랙션·애니메이션 | 의미 있는 모션 + 성능 유지 | 호버/전환 효과 있음 | 효과 없음 또는 과도함 |
| D2 | 콘텐츠 큐레이션 | AI/알고리즘 기반 맞춤 추천 | 카테고리 분류 | 시간순 나열만 |
| D3 | 개인화·AI 활용 | 챗봇/추천/맞춤 경험 | 기본 추천 | 개인화 없음 |
| D4 | 차별화 요소 | 업종 내 유일한 기능/경험 | 경쟁사 대비 1~2개 차이 | 경쟁사와 동일 |
| D5 | 트렌드 반영 | 2025~2026 트렌드 3개+ 적용 | 1~2년 이내 트렌드 | 3년+ 이전 디자인 |

## 3. 업종별 가중치 조정

특정 업종에서는 영역별 가중치를 조정합니다:

| 업종 | 디자인 | 기능성 | 사용성 | 혁신성 | 이유 |
|------|--------|--------|--------|--------|------|
| 공공/기관 (A) | 20 | 25 | **30** | 25 | 접근성·사용성 최우선 |
| 커머스 (D) | 20 | **30** | **30** | 20 | 기능+전환율 중심 |
| 문화/예술 (E) | **30** | 20 | 20 | **30** | 비주얼+경험 중심 |
| 패션/라이프 (H) | **30** | 20 | 25 | 25 | 룩북·비주얼 중심 |
| 금융/법률 (L) | 20 | **30** | **30** | 20 | 신뢰·보안·사용성 |

> 가중치 미지정 업종은 기본값(균등 25) 적용.

## 5. 종합 판정

| 총점 | 판정 | 활용 방침 |
|------|------|----------|
| 80~100 | **핵심 레퍼런스** | 디자인 방향의 주요 참조 대상 |
| 60~79 | **부분 참조** | 특정 영역(컬러/레이아웃 등)만 참조 |
| 40~59 | **참고 수준** | 트렌드 파악 용도 |
| 0~39 | **반면교사** | 피해야 할 패턴으로 기록 |
