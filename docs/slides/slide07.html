Tech Stack

RUNTIME

Node.js + Express

Lightweight HTTP server. RESTful API endpoints. SSE for real-time dashboard updates.

DATABASE

SQLite (WAL Mode)

Zero-config embedded DB. WAL enables concurrent reads during writes. Single-file backup.

AI INTEGRATION

MCP Server + Claude API

Model Context Protocol for tool calling. Opus for creation, Sonnet for review. Structured JSON I/O.

VERSION CONTROL

Git Hooks + File System

Pre-commit validation. JSON schema checks. Automated deployment pipeline.

DEV Environment

Claude Code CLI - Interactive development, skill testing, pipeline prototyping

PRD Environment

API Daemon - Automated execution, scheduled tasks, production workflows
