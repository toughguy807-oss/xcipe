04

Solution: 6-Layer Architecture

ELUO SYS platform designed as layered, modular system

1

Scheduling Layer

Cron jobs, pipeline orchestration, task queue

2

Interface Layer

Web dashboard, API endpoints, CLI bridge

3

State Layer

SQLite WAL, JSON config, session management

4

Execution Layer

Claude API, MCP servers, skill runtime

5

Knowledge Layer

Project memory, design principles, traceability DB

6

Foundation Layer

Node.js runtime, file system, Git version control

On-Premise First

Data sovereignty. Client strategies, specs, competitor analysis stored locally.

JSON Interface

Agents communicate via JSON files. Schema validation ensures zero parse failures.

Production != Review

Separate models: opus for creation, sonnet for review. Eliminates self-review bias.
